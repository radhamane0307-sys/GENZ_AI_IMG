
# 🚀 GENZ AI Image Generator
MERN AI Image Generator

A full-stack MERN AI Image Generator web app that allows users to generate images based on prompts.

## 🔥 Features

* 🔐 User Authentication (JWT)
* 🎨 AI Image Generation (Unsplash API)
* ☁️ Cloud Storage (Cloudinary)
* 📥 Download Generated Images
* ⚡ Fast & Responsive UI

## 🛠️ Tech Stack

* Frontend: React (Vite)
* Backend: Node.js + Express
* Database: MongoDB
* APIs: Unsplash
* Authentication: JWT

## 🚀 How to Run

### 1. Clone repo

```bash
git clone https://github.com/radhamane0307-sys/GENZ_AI_IMG.git
```

### 2. Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### 3. Add .env file in server

```
MONGO_URI=your_mongo_url
JWT_SECRET=your_secret
UNSPLASH_ACCESS_KEY=your_key
```

### 4. Run project

```bash
npm run dev
```
